# GWFD-010105用户面地址分配特性概述

## 适用NF

PGW-U、UPF

## 定义

['UE IP地址是UE在网络中进行通信和寻址的唯一标识，用户上网过程中必须被分配IP地址才可正常通信。用户面地址分配是指UDG本地为UE分配地址，可提供三种类型的IP地址：IPv4地址、IPv6地址、IPv4v6双栈地址。', '由于UDG为用户面网关，处于分组核心网和外部DN交换的位置，因此UDG在为UE分配IP地址的同时，还可生成下行路由用于下行报文的回程寻址。']

## 客户价值

[['表头：', '受益方', '受益描述'], ['第1行：', '运营商', '用户面地址分配由于是在网关上进行的，因此在地址管理和运维操作上相对更加便利。 地址规划上考虑UDG整机地址不冲突。 在网关上可规划用户地址段生成子段下行路由，避免可能存在的大量零散主机路由发布到骨干网后对网络造成冲击。'], ['第2行：', '用户', '用户使用分配的IP地址，可与网络中其他实体正常通信以及被寻址，成功接入PDN/DN网络，享用移动通信网络提供的丰富多样的数据业务和语音业务。']]

## 应用场景

运营商为用户规划动态IP地址，且不期望单独部署分配IP地址的设备。此时，可以开启UDG用户面地址分配功能，使用UDG本地为用户分配动态IP地址，使得运营商地址管理和运维操作更加便利，也节约了运营成本。

[['表头：', '交互类型', '相关特性', '控制项名称', '交互说明'], ['第1行：', '依赖', 'GWFD-010224 N4接口特性概述', '无', '5G场景下，UDG向C面上报用户面地址分配能力以及C面指示UDG进行地址分配依赖于N4接口的交互。使用本特性需要使用N4接口特性。'], ['第2行：', '依赖', 'GWFD-010233 Sxb接口', '无', '4G场景下，UDG向C面上报用户面地址分配能力以及C面指示UDG进行地址分配依赖于Sxb接口的交互。使用本特性需要使用Sxb接口特性。']]

## 对系统的影响

本特性对系统无影响。

## 应用限制

由于该特性在N4/Sxa/Sxb接口对PFCP协议有流程和信元的私有扩展，因此要求对接的C面网元必须为华为配套设备。

## 原理概述

['UDG通过N4/Sxa/Sxb接口的偶联建立响应消息中的私有扩展域告知C面本地分配地址能力，C面获取UDG该能力。', '用户激活向C面发起会话建立请求，申请分配IP地址，C面决策由U面分配指定类型的IP地址（IPv4、IPv6或IPv4v6双栈）。在N4/Sxa/Sxb接口通过PFCP会话建立请求消息（PFCP Session Establishment Request）指示UDG分配地址。UDG根据指示为用户分配地址，并通过会话建立响应消息将分配的IP地址传递给UE，UE获取IP地址后便可发起上下行数据业务。']
