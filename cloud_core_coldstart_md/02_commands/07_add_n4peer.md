# 添加N4对端（ADD N4PEER）（ADD N4PEER）

## 命令功能

['适用NF：SMF、UPF', '该命令用于配置N4接口对端节点信息，包括对端地址、端口、心跳周期和关联名称。']

## 注意事项

['对端IP地址应与承载口路由可达。', '若启用防火墙或ACL，需放通PFCP相关端口。', '心跳周期和超时阈值需结合网络抖动情况合理配置。']

## 操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数说明

[['表头：', '参数标识', '参数名称', '参数说明'], ['第1行：', 'PEERNAME', '对端名称', '必选参数。用于标识N4对端。'], ['第2行：', 'PEERIP', '对端IP地址', '必选参数。指定PFCP对端地址。'], ['第3行：', 'LOCALIP', '本端源地址', '可选参数。指定本端发送PFCP消息使用的源地址。'], ['第4行：', 'HBPERIOD', '心跳周期', '可选参数。用于配置PFCP心跳检测周期。'], ['第5行：', 'RETRYTHRESHOLD', '超时阈值', '可选参数。定义连续超时次数。']]

## 使用实例

['在SMF上新增UPF对端upf01：', 'ADD N4PEER:PEERNAME="upf01",PEERIP="10.10.10.2",LOCALIP="10.10.10.1",HBPERIOD=30,RETRYTHRESHOLD=3;']
