# 添加地址池（ADD POOL）（ADD POOL）

## 命令功能

['适用NF：UPF、PGW-U', '该命令用于创建用户地址池对象，并定义地址池类型、归属VPN以及是否用于本地地址分配。']

## 注意事项

['该命令执行后立即生效。', '同一地址池名称在系统内必须唯一。', '本地地址池用于用户面本地分配地址，外部地址池通常用于与外部分配系统协同。', '配置地址池后，需继续配置地址段并在DNN/APN侧进行绑定，否则地址池不能直接为用户分配地址。']

## 操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数说明

[['表头：', '参数标识', '参数名称', '参数说明'], ['第1行：', 'POOLNAME', '地址池名称', '必选参数。字符串类型，长度1～79字节，用于唯一标识地址池。'], ['第2行：', 'POOLTYPE', '地址池类型', '必选参数。枚举类型，可选LOCAL或EXTERNAL。LOCAL表示本地地址池，EXTERNAL表示外部地址池。'], ['第3行：', 'VPNNAME', 'VPN名称', '可选参数。用于约束地址池所在VPN域，未配置时按默认VPN处理。'], ['第4行：', 'DESC', '描述', '可选参数。用于记录地址池用途、归属业务或维护备注。']]

## 使用实例

['创建本地地址池pool_ipv4：', 'ADD POOL:POOLNAME="pool_ipv4",POOLTYPE=LOCAL,VPNNAME="vpn_internet";', '创建外部地址池pool_ext：', 'ADD POOL:POOLNAME="pool_ext",POOLTYPE=EXTERNAL,VPNNAME="vpn_enterprise";']
