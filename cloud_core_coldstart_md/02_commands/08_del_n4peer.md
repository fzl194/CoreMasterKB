# 删除N4对端（DEL N4PEER）（DEL N4PEER）

## 命令功能

['适用NF：SMF、UPF', '该命令用于删除已配置的N4对端。执行前应确认该对端未承载生产会话。']

## 注意事项

['若对端仍有现网会话，删除后可能导致新建会话失败或保护切换。', '建议先做流量摘除或切换，再清理对端配置。']

## 操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数说明

[['表头：', '参数标识', '参数名称', '参数说明'], ['第1行：', 'PEERNAME', '对端名称', '必选参数。指定待删除的N4对端。']]

## 使用实例

['删除N4对端upf01：', 'DEL N4PEER:PEERNAME="upf01";']
