# 网络切片技术

网络切片（Network Slicing）是5G系统的核心特性之一，它允许运营商在同一物理网络基础设施上构建多个端到端的虚拟网络，每个切片可以独立配置网络功能、资源和拓扑，以满足不同业务场景的差异化需求。

## 网络切片的概念

一个**网络切片**由一组网络功能实例及其所需的资源组成，包括RAN侧的切片资源和核心网侧的NF实例。不同切片之间在逻辑上隔离，可以根据业务需求提供不同的SLA（Service Level Agreement），如不同的带宽、时延、安全等级和可靠性。

网络切片贯穿5G系统的三个层面：

- **UE侧**：UE通过携带S-NSSAI请求接入特定切片
- **RAN侧**：gNB根据切片标识配置不同的无线资源
- **核心网侧**：5GC为不同切片选择专用的AMF、SMF、UPF实例

## S-NSSAI

**S-NSSAI（Single Network Slice Selection Assistance Information）** 是3GPP定义的网络切片标识，用于唯一标识一个网络切片。每个S-NSSAI由两部分组成：

| 组成部分 | 说明 |
|---------|------|
| **SST（Slice/Service Type）** | 切片/服务类型，标识切片的业务类型。长度为8 bit（标准值范围0~127），是必选字段 |
| **SD（Slice Differentiator）** | 切片区分符，可选字段（长度24 bit），用于在同一SST下区分多个切片实例 |

S-NSSAI分为两种：

- **标准化S-NSSAI**：SST值为3GPP全球统一规定，在所有PLMN中含义一致。
- **非标准化S-NSSAI**：SST值由运营商自定义，仅在归属PLMN中使用。此类S-NSSAI在漫游场景中不会被传递到拜访网络。

## 标准化SST值

3GPP TS 23.501定义了以下标准化SST值：

| SST值 | 切片类型 | 说明 |
|-------|---------|------|
| **1** | **eMBB**（Enhanced Mobile Broadband） | 增强移动宽带，面向高速率业务（如高清视频、VR/AR） |
| **2** | **URLLC**（Ultra-Reliable Low-Latency Communication） | 超可靠低时延通信，面向工业自动化、远程控制、自动驾驶 |
| **3** | **mMTC**（Massive Machine Type Communication） | 海量物联网，面向大规模传感器、智能抄表等低速率高连接密度场景 |
| **4** | **V2X**（Vehicle-to-Everything） | 车联网（Rel-16新增） |

运营商还可以使用值128~255定义自有的切片类型。

## 切片选择流程

网络切片的选择和关联发生在UE注册和PDU会话建立阶段：

1. **注册阶段**：
   - UE在Registration Request中携带Requested NSSAI（希望使用的切片列表）。
   - AMF查询**NSSF（Network Slice Selection Function）**，NSSF根据UE的签约切片（Subscribed S-NSSAI，存储于UDM）、Requested NSSAI和运营商策略，确定允许的S-NSSAI（Allowed NSSAI）。
   - NSSF还可能为AMF推荐合适的NSI（Network Slice Instance）或目标AMF Set，必要时触发AMF重选。

2. **PDU会话建立阶段**：
   - UE在会话建立请求中携带目标S-NSSAI。
   - SMF根据S-NSSAI选择对应的UPF，确保用户面在正确的切片实例内建立。
   - 不同切片的SMF/UPF可以是专用的实例，也可以共享，取决于运营商部署策略。

3. **RAN侧**：
   - UE通过RRC信令将S-NSSAI告知gNB。
   - gNB根据S-NSSAI配置对应的无线资源（如PRB分配、调度策略），并将UE路由到正确的核心网AMF。

## 切片与网络功能选择的关系

切片信息直接影响多个NF的选择决策：

| 网络功能 | 切片影响 |
|---------|---------|
| **AMF** | 支持特定切片的AMF集合可能不同，NSSF可触发AMF重选 |
| **SMF** | SMF选择需要匹配S-NSSAI，不同切片可能使用专用SMF |
| **UPF** | UPF选择受S-NSSAI和DNN共同影响，切片间UPF可隔离 |
| **PCF** | PCF可能按切片提供差异化的策略规则 |
