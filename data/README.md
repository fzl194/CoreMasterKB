# Test Data Area

This directory stores hand-built corpora and question sets used to validate the
Knowledge Mining -> Knowledge Assets -> Agent Serving contract.

Current data sets:

- `m1_contract_corpus/`: first-layer M1 contract corpus. It is intentionally
  small and based on the existing `cloud_core_coldstart_md/` materials.
- `m1_realistic_corpus/`: realistic M1 corpus shaped by the sample corpus
  templates and grounded in public 3GPP, free5GC, and Open5GS sources.
